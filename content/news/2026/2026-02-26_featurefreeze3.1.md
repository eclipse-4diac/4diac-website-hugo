The 3.1 release introduces several architectural and usability enhancements:
* **4diac IDE:** Improved Struct Member Access for `F_MOVE` blocks, support for Folder Refactoring, and a refined user interface including drop shadows and optimized Dark Mode support.
* **4diac FORTE:** A modularized IEC 61131-3 function library, memory optimizations for FreeRTOS targets, and expanded OPC UA Alarm handling for SIFB-based implementations.

The final release is scheduled for March 11, 2026. Detailed information on these changes is available on the [New and Noteworthy pages](/4diac/new-and-noteworthy/3.1/).

Testing of the current development branches is encouraged to assist in the final stabilization phase. Please report any issues via our GitHub issue trackers.
