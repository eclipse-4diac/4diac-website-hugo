## A Major Milestone for the 4diac Ecosystem

The Eclipse 4diac 3.0 release marks the largest update we have ever delivered.
Over the course of this release cycle, we completed:

 - 5974 commits on the 4diac IDE
 - 1537 commits on 4diac FORTE
 - A brand-new component: 4diac FBE, our flexible build environment

In the final weeks before release we focused heavily on stabilization, fixing a large number of issues and polishing the user experience across all components.

## Whatâ€™s New in Eclipse 4diac 3.0 (Short Overview)

This release brings major improvements across engineering, runtime execution, and build workflows:

 - **4diac IDE**
   - Fully reworked IEC 61131-3 Structured Text editors
   - New ST interpreter for testing and debugging FBs directly in 4diac IDE
   - Improved IEC 61499 project validation with repair options
   - New library & package system, named constants, IEC 61131-3 functions, and VAR_IN_OUT
   - Simplified deployment & monitoring with saved configurations
 - **4diac FORTE**
   - ARRAYs with arbitrary and dynamic boundaries
   - Improved IEC 61131-3 compliance and function coverage
   - Better handling of ANY, STRING, and casts
   - Added variadic functions, high-resolution clocks, and a standard-conformant FOR loop
   - Now fully based on C++20
 - **4diac FBE (new in 3.0)**
   - Automatic, dependency-free builds of 4diac FORTE
   - Manage multiple 4diac FORTE configurations for diverse targets
   - Beta-level 4diac IDE integration available
