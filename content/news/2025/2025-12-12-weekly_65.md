With this major milestone reached, the focus now shifts to consolidation, bug fixes, and incremental improvements across the entire Eclipse 4diac ecosystem.



## Enhancements and Fixes in 4diac IDE

- **ST, parsing, and type handling**
  - Validate ST expressions for side effects in the parser
  - Fix parsing of query type responses for data types, adapters, and global constants
  - Gracefully handle empty type hash attributes
  - Add literal types to XMI export

- **Copy, refactoring, and model consistency**
  - Fix missing initial values and comments when copying cached interfaces
  - Fix refactoring of initial values inside structs by aligning with ST handling
  - Avoid duplicate error markers for elements mapped from applications
  - Include error marker pins when searching pins by name

- **UI, layout, and usability improvements**
  - Improve default block size parameters for better look & feel
  - Adjust initial sizes of groups and subapps
  - Fix instance creation issues in resources and palette filtering

- **Validation, robustness, and bug fixes**
  - Fix exception in dynamic contract checker
  - Fix issues in instance creation and type queries
  - Add SERVER_0 and CLIENT_0 defaults

- **Build, CI, and maintenance**
  - Update 3.0.x branch feature version to 3.0.1
  - Update GitHub Actions (setup-java, checkout)
  - Update IDE internal help pages and documentation sources


## Enhancements and Fixes in 4diac FORTE

- **Build and CI maintenance**
  - Update cpp-linter GitHub Action
  - Upgrade GitHub Actions checkout to version 6

- **Code cleanup**
  - Clean up timeout block implementations to better align with exported behavior


## Enhancements and Fixes in 4diac Documentation
- **Structure and navigation**
  - Move advanced features to a dedicated section
  - Add anchors for improved IO configuration navigation


## Enhancements and Fixes in 4diac FBE
- **Toolchain and release alignment**
  - Update toolchain submodules and packaging fixes
  - Align 4diac FORTE submodule with the 3.0 release
  - Add release build configurations for 4diac FORTE


Many thanks to all contributors for their work and their feedback!
