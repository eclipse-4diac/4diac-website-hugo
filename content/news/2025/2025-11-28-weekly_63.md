## Enhancements and Fixes in 4diac IDE

- **Monitoring & Visibility**
  - Filter monitoring to block FB network elements only.
  - Fix visibility handling for `VarInOut` outputs after restart.

- **Member Access Pin Handling**
  - Correct handling of member access pins when saving and parsing.
  - Fix duplicate connections created for output member access pins.
  - Check if connecting to a member access pin is valid.
  - Support full type names in type prefixes for generic variables.
  - Support full type names for struct and enum values.
  - Rework visibility storage for Member Access pins.
  - Correct visibility saving for MemberAccessPins in XML.
  - Fix tooltip and max string size for member access pins.
  - Show initial values for member access pins.

- **Structured Text (ST) Improvements**
  - Fix ST documentation hover for variables.
  - Fix ST documentation hover for functions.
  - Fix compilation issues in ST quickfix providers.

- **Refactoring & Copy Handling**
  - Remove unintended renaming during copy refactoring.
  - Improve and clean up copy refactoring.
  - Ensure file copy happens during refactoring.
  - Fix loading of copy processor participants.
  - Fix copy nature IDs and participant behavior.
  - Use `Objects.equals` to avoid NPEs.
  - Add copy refactoring to system explorer:
    - Update package names when types are copied.
    - Add “copy package name” action.

- **Connections & Mapping**
  - Fix broken connection repair handler to validate selection types.
  - Fix validation of InOut variables inside subapps.
  - Handle mapping of connection attributes (#1542).
  - Fix exporting of large files.
  - Correct adapter FB plug/socket interfaces on create & update.

- **Deployment & Error Handling**
  - Improve error messages for parameter write failures.
  - Adjust deployment exception handling to reveal root cause.
  - Add configurable maximum total request size for deployment.
  - Log failed write requests.
  - Directly update expected type in embedded field editor.

- **General Fixes & Cleanup**
  - Fix focus handling for `InitialValueStructuredCellEditor`.
  - Fix JUnit build failures with Eclipse 2025-12; pin to JUnit 5 and update Tycho.
  - Code cleanup for Sonar-reported issues.
  - Clean up `FordiacResourceChangeListener`.
  - Upgraded Xtend Maven plugin to 2.41.0.



## Enhancements and Fixes in 4diac FORTE

- **Error & Response Handling**
  - Ensure long error responses are correctly embedded in overall responses.

- **Type Hash Diagnostics**
  - On mismatched type hashes, return the available type together with its actual hash.


Many thanks to all contributors for their work and their feedback!
