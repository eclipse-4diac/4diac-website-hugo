4diac FBE builds 4diac FORTE fully automatically, no manual downloads or installs of third-party software needed. It manages multiple 4diac FORTE configurations for multiple target machines and operating systems on a single developer machine.

It supports x86_64 Linux, x86_64 Windows and ARM64-based MacOS machines as developer machines, and it can cross-build 4diac FORTE binaries for pretty much all relevant MacOS, Windows and Linux targets, including single-board computers like the Raspberry Pi and legacy hardware.

---

Give it a test drive at https://github.com/eclipse-4diac/4diac-fbe/releases/tag/3.0-beta1

Download the ZIP file, extract it, and run `compile.cmd` to create a default full-featured FORTE binary that has enabled more or less all supported communication protocols (like OPC-UA, MQTT and others).

And yes, that single ZIP file with its `compile.cmd` script works on Windows, Linux and MacOS. Read the instructions in the included README file for all the details.
